# Практична робота: SELECT у PostgreSQL

## Контекст
У базі даних кінотеатру зберігається інформація про фільми в таблиці `films`.

Раніше до таблиці було додано такий запис:

INSERT INTO films (title, genre, duration_minutes, rating, is_available)
VALUES ('Matrix', 'Sci-Fi', 136, 8.7, true);

Тепер потрібно навчитися **переглядати дані** з таблиці.

---

## Завдання 1
Виведіть **усі фільми** з таблиці `films`.

---

## Завдання 2
Виведіть **тільки назву та жанр** усіх фільмів.

---

## Завдання 3
Виведіть фільми, які:
- доступні для перегляду (`is_available = true`)

---

## Завдання 4
Виведіть фільми, тривалість яких **більше 120 хвилин**.

---

## Підказка
Для вибірки даних використовується команда:

SELECT ... FROM ...
