## Контекст
Уявіть, що ви створюєте базу даних для кінотеатру.

У базі даних зберігається інформація про фільми, які зараз показують у кінотеатрі.

Для кожного фільму потрібно зберігати:
- унікальний ідентифікатор
- назву фільму
- жанр
- тривалість (у хвилинах)
- рейтинг фільму
- чи доступний фільм для перегляду зараз

# Завдання на INSERT INTO

## Завдання 1
Додайте **один фільм** у таблицю `films`.

Обовʼязково заповніть всі поля, крім `id`.
