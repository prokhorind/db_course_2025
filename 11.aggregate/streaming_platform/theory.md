# SQL Агреговані функції

Агреговані функції використовуються для обчислення значення на основі кількох рядків таблиці.
Вони повертають одне підсумкове значення.

--------------------------------------------------

ОСНОВНІ АГРЕГОВАНІ ФУНКЦІЇ

COUNT()  – рахує кількість рядків
SUM()    – рахує суму значень
AVG()    – обчислює середнє значення
MAX()    – знаходить найбільше значення
MIN()    – знаходить найменше значення

--------------------------------------------------

COUNT()

Рахує кількість рядків у таблиці.

SELECT COUNT(*)
FROM users;

--------------------------------------------------

SUM()

Рахує суму значень у стовпці.

SELECT SUM(amount)
FROM payments;

--------------------------------------------------

AVG()

Обчислює середнє значення.

SELECT AVG(rating)
FROM watch_history;

--------------------------------------------------

MAX() та MIN()

SELECT MAX(rating)
FROM watch_history;

SELECT MIN(rating)
FROM watch_history;

--------------------------------------------------

GROUP BY

GROUP BY використовується для групування рядків за певним стовпцем.

Після цього агреговані функції застосовуються до кожної групи окремо.

Приклад:

Скільки користувачів кожного типу підписки.

SELECT subscription_type, COUNT(*)
FROM users
GROUP BY subscription_type;

--------------------------------------------------

GROUP BY + JOIN

Можна групувати дані після об'єднання таблиць.

SELECT m.title, COUNT(*)
FROM watch_history w
JOIN movies m ON m.movie_id = w.movie_id
GROUP BY m.title;

Цей запит показує скільки разів дивилися кожен фільм.

--------------------------------------------------

HAVING

HAVING використовується для фільтрації груп після GROUP BY.

Важливо:

WHERE  – фільтрує рядки
HAVING – фільтрує групи

Приклад:

Фільми які дивилися більше одного разу.

SELECT movie_id, COUNT(*)
FROM watch_history
GROUP BY movie_id
HAVING COUNT(*) > 1;

--------------------------------------------------

ПОРЯДОК ВИКОНАННЯ SQL ЗАПИТУ

1. FROM
2. JOIN
3. WHERE
4. GROUP BY
5. HAVING
6. SELECT
7. ORDER BY

--------------------------------------------------

ПРИКЛАД АНАЛІТИЧНОГО ЗАПИТУ

SELECT m.title, COUNT(*) AS views
FROM watch_history w
JOIN movies m ON m.movie_id = w.movie_id
GROUP BY m.title
HAVING COUNT(*) > 1
ORDER BY views DESC;

Цей запит знаходить найпопулярніші фільми.
